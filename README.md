# My Flow Money

A premium local-first AI-powered expense tracker built with Next.js 15.

## Why this app

My Flow Money is designed for instant personal finance tracking with zero backend requirement.

- Local-first with IndexedDB (Dexie)
- Offline-first PWA
- No-login usage
- AI-like financial insights (rule-based, fully local)
- Optional cloud backup payload generation

## Stack

- Next.js 15 App Router + TypeScript
- Tailwind CSS + reusable shadcn-style UI components
- Zustand for app state orchestration
- Dexie.js for local database
- React Hook Form + Zod for form validation
- Framer Motion for smooth animations
- Recharts for analytics visualizations
- next-pwa for installable/offline app behavior

## Core Features

- Dashboard: balance, income/expense, budget remainder, trend chart, health score, smart recap
- Transactions: add/edit/delete, search, type filter, recurring flag, quick add modal
- Smart categories with colors and analytics support
- Wallet system: cash, bank, e-wallet, credit card
- Budget monitor with warning progress
- Analytics page: weekly/monthly trends + wallet comparison
- Local AI engine: overspending alerts, habits, month-to-month recap, subscription detection
- Backup system: export/import JSON and cloud-sync payload preparation
- Command palette (`Ctrl/Cmd + K`) and quick-add shortcut (`Ctrl/Cmd + N`)

## PWA Notes

- Install prompt available in supported browsers
- Service worker generated by `next-pwa`
- Offline document fallback route: `/offline`

## Cloud Backup Model

No database is used.

Cloud backup is optional and works through JSON snapshots:

1. Export data to JSON
2. Upload JSON to storage provider (Supabase Storage / Google Drive) with your own SDK integration
3. Restore by importing JSON

## Deployment
